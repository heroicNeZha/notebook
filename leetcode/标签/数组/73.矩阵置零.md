# 矩阵置零
## 初解
##### 想法
使用额外空间记录应当置零的行号和列号
##### 算法
```java
class Solution {
    public void setZeroes(int[][] matrix) {
        if (matrix == null || matrix.length == 0)
            return;
        int m = matrix.length;
        int n = matrix[0].length;
        boolean[] lineFlag = new boolean[m];
        boolean[] colFlag = new boolean[n];
        // 标志位
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (matrix[i][j] == 0) {
                    lineFlag[i] = true;
                    colFlag[j] = true;
                }
            }
        }
        // 置零
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (matrix[i][j] != 0 && (lineFlag[i] || colFlag[j])) {
                    matrix[i][j] = 0;
                }
            }
        }
    }
}
```
##### 复杂度分析
时间复杂度：O(mn)  
空间复杂度：O(m+n)

## 最优解
##### 想法
通过第一行记录为零的行号列号
##### 算法
```java
class Solution {
    public void setZeroes(int[][] matrix) {
        if (matrix == null || matrix.length == 0) return;
        int m = matrix.length;
        int n = matrix[0].length;
        //分别是首列与首行存0情况
        boolean flagN = false;
        boolean flagM = false;
        for (int i = 0; i < m; i ++) {
            //判断第一列原先是否存在0
            if (matrix[i][0] == 0) flagN = true;
        }
        for (int j = 0; j < n; j ++) {
            //判断第一行原先是否存在0
            if (matrix[0][j] == 0) flagM = true;
        }
        //判断非首行首列的0情况，在首行首列进行更改
        for (int i = 1; i < m; i ++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][j] == 0) {
                    matrix[i][0] = 0;
                    matrix[0][j] = 0;
                }
            }
        }
        //遍历非首行首列，以首行首列的0为标志更改
        for (int i = 1; i < m; i ++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][j] != 0 && (matrix[i][0] == 0 || matrix[0][j] == 0)) {
                    matrix[i][j] = 0;
                }
            }
        }
        //然后解放首行首列
        if (flagN) {
            for (int i = 0; i < m; i ++) {
                matrix[i][0] = 0;
            }
        }
        if (flagM) {
            for (int j = 0; j < n; j ++) {
                matrix[0][j] = 0;
            }
        }
    }
}
```
##### 复杂度分析
时间复杂度：O(mn)  
空间复杂度：O(1)
