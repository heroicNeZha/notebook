# 快速排序

### 将第一个元素作为枢轴

1. 最优和平均时间复杂度O(nlgn)
2. 最坏情况为排好序的情况O(n^2)
3. 代码如下
    ```java
    public int division(int[] nums, int left, int right) {
        int flag = nums[left];
        while (left < right) {
            while (left < right && nums[right] >= flag) {
                right--;
            }
            nums[left] = nums[right];
            while (left < right && nums[left] <= flag) {
                left++;
            }
            nums[right] = nums[left];
        }
        nums[left] = flag;
        return left;
    }

    public void quicksort(int[] nums, int start, int end) {
        if (start < end) {
            int base = division(nums, start, end);
            quicksort(nums, start, base - 1);
            quicksort(nums, base + 1, end);
        }
    }
    ```
### 中间元素作为枢轴
1. 对于近似有序来说,选择中间元素更有效率一点.
2. **仍然很慢**
3. 代码如下:
    ```java
    public int division(int[] nums, int left, int right) {
        int flag = nums[(left + right) / 2];
        nums[(left + right) / 2] = nums[left];
        nums[left] = flag;
        while (left < right) {
            while (left < right && nums[right] >= flag) {
                right--;
            }
            nums[left] = nums[right];
            while (left < right && nums[left] <= flag) {
                left++;
            }
            nums[right] = nums[left];
        }
        nums[left] = flag;
        return left;
    }

    public void quicksort(int[] nums, int start, int end) {
        if (start < end) {
            int base = division(nums, start, end);
            quicksort(nums, start, base - 1);
            quicksort(nums, base + 1, end);
        }
    }
    ```
