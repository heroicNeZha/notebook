最优 投票算法：
```java
class Solution {
    public int majorityElement(int[] nums) {
        Stack<Integer> stack = new Stack<>();
        for (int i = 0; i < nums.length; i++) {
            if (stack.empty() || stack.peek() == nums[i]) {
                stack.push(nums[i]);
            } else {
                stack.pop();
            }
        }
        return stack.pop();
    }
}
```
