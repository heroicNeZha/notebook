# Vue

### Vue环境介绍
1. node.js和npm可以通过版本管理工具nvm来控制,可以用cnpm、别名和淘宝镜像源来加速下载.
2. sudo npm install -g @vue/cli
3. vue create my-app
4. code my-app
5. 安装vetur
6. chrome 安装插件 vuejs devtools
